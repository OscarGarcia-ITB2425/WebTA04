Gestor de Tareas por Terminal

Fecha de inicio: 23/11/2024

Tecnologías utilizadas: VSC, Python

Descripción

Un gestor de tareas sencillo para utilizar en la terminal de cualquier ordenador o dispositivo que tenga Python (versión 3.8 o superior). Este proyecto me permitió mejorar mis habilidades en programación orientada a objetos y manejo de archivos.

Características destacadas

  • Agregar, listar y eliminar tareas.
  • Persistencia de datos mediante archivos.
  • Interfaz de línea de comandos intuitiva.
