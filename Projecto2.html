Calculadora Gráfica

Fecha de inicio: 18/5/2022

Tecnologías utilizadas: VSC, Python, Tkinter

Descripción: Creación de una calculadora gráfica básica utilizando Python y la biblioteca Tkinter. Este proyecto me ayudó a entender la construcción de interfaces gráficas de usuario (GUI) y eventos en Python.

Características destacadas:

  • Operaciones aritméticas básicas.
  • Interfaz amigable y fácil de usar.
  • Gestión de errores de entrada.
